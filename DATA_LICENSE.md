# Data license

The analysis-ready tabular data in this repository are provided to support transparency and reproducibility of the associated manuscript:

**Satellite Embeddings as Complementary Predictors of Standing Dead-Tree Volume Fraction in Boreal Forests**

## Licensed data

Unless otherwise stated, the analysis-ready tabular data provided in the `data/` folder are made available under the **Creative Commons Attribution 4.0 International License (CC BY 4.0)**.

This means that others may use, share, and adapt the public analysis-ready data, provided that appropriate credit is given to the authors and the associated repository/manuscript is cited.

## Data not included

The following materials are not redistributed in this repository:

- exact field-plot coordinates;
- original plot/site names;
- raw field inventory records containing sensitive location information;
- raw canopy-height model rasters;
- raw aerial RGB-NIR imagery;
- raw AlphaEarth satellite-embedding rasters.

Raw spatial and raster datasets remain subject to the licenses and access conditions of their original data providers.

## Suggested citation

If you use the data or code in this repository, please cite both the associated manuscript and this GitHub repository.
